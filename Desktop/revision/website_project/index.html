CSS is a language that describes the style of an HTML document.This tutorial will teach you CSS from basic to advanced.The difference between an ID and a class is that an ID can be used to identify one element, whereas a class can be used to identify more than one.

You can also define your own selectors in the form of class and ID selectors. The benefit of this is that you can have the same HTML element, but present it differently depending on its class or ID.
